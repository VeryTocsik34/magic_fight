# 🔮 Битва магов

Волшебная аркадная игра, в которой два мага сражаются друг с другом с помощью магических сфер. Поддерживается игра против компьютера или вдвоём на одном устройстве.

## 🎮 Геймплей

Каждый игрок управляет магом, который может:
- **Двигаться** по арене
- **Заряжать магическую сферу** (чем дольше зарядка — тем больше урон)
- **Атаковать** противника выпущенной сферой

### Управление

| Действие | Игрок 1 | Игрок 2 (2P режим) |
|----------|---------|---------------------|
| Влево    | A       | ←                   |
| Вправо   | D       | →                   |
| Присесть | S       | ↓                   |
| Зарядка  | Пробел  | Right Ctrl          |

> 💡 В режиме игры против компьютера (1P) можно управлять только левым магом. ПК-противник атакует автоматически.

## 🔧 Особенности

- **PVE режим** — сражение с компьютерным противником
- **PVP режим** — дуэль двух игроков на одном устройстве
- **Экспериментальный режим жестов** — управление с помощью камеры (требует доработки модуля `gesture.py`)
- Различные классы магов (огненный, земляной, молниеносный)
- Система здоровья и полоски заряда атаки
- Анимированные персонажи
